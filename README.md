# Mga Kuwento at Aral ng 25 Propeta (A.S.)

Isang kumpletong gabay sa buhay, pagkatao, at mahahalagang aral ng 25 Propeta ng Allah (A.S.) ayon sa Qur’an. Basahin ang kanilang mga kuwento at matutunan ang pananampalataya, katapangan, at kabutihan sa bawat kabanata.

## Features
- 25 detalyadong kabanata, bawat Propeta ay may buhay, aral, at sanggunian sa Qur’an
- Dark / Light mode
- Mobile-friendly at responsive

## Pages
- `index.html` – Home page at Table of Contents
- `pages/kabanata1.html` hanggang `pages/kabanata25.html` – Bawat kabanata ng Propeta

## How to host
1. Create a new GitHub repository
2. Copy all files into the repo
3. Enable GitHub Pages from the repository settings
4. Visit the GitHub Pages URL to see your website
